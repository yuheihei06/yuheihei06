### Hi there, I'm 简朴 👋

> *"I'm building a universe to help people see their own cognitive boundaries."*

我是一名大二学生，正在用 AI 辅助做一个以“信息茧房可视化”为核心概念的 3D 沉浸式宇宙茧房项目。

### 🚀 当前主推项目：InfoCocoon-Galaxy

**简介**：是受无人深空启发开发的一个信息茧房可视化项目。目的是基于现在人们受到信息茧房的包裹越来越严重，加之互联网算法过滤泡的影响，使得人们总是在自己的小圈子故步自封，我做的这个项目是给人们一个可以破圈，去了解其他方面的渠道。

**✨ 核心体验亮点**：
- 🌌 **四级宇宙探索**：从星座到卫星，层层深入每一个未知的兴趣领域。
- 🚀 **星系内部体验太空飞行**：已经设计了星系内部行星的运转和飞船的移动，可供用户自由在宇宙探索。
- 💥 **引导身份塌缩**(开发中)：选择极端茧房身份，进入被固有认知包裹的宇宙，引导结束后，整个星系在眼前**塌缩、湮灭、重生**，那些你曾以为坚固的认知连线，原来只是一座牢笼。
- 🌏**可时时更新热点的行星**(开发中)；破圈的核心，给用户选择其他圈子的渠道。
- 🧠 **AI 克隆体**（开发中）：一个“知音但无茧房”的数字镜像，既能提供情绪价值，又能温柔地帮你破圈。

**💡 设计哲学**  
> *"信息茧房的打破，不应该靠说教，而应该靠一次亲眼目睹的毁灭。"*

**📅 当前进度**：
- ✅ 前端四层宇宙界面框架搭建完毕，已经完成 MVP 主链路，并已具备从星系图探索、全息预览、星系内部飞行，到进入行星观察界面的完整体验。
- 🚧 正在开发：星系内部的关联
- 🎯 下一个里程碑：MVP 上线，开放内测
