**一句话简介**：是受无人深空启发开发的一个信息茧房可视化项目。目的是基于现在人们受到信息茧房的包裹越来越严重，加之互联网算法过滤泡的影响，使得人们总是在自己的小圈子故步自封，我做的这个项目是给人们一个可以破圈，去了解其他方面的渠道。

**✨ 核心体验亮点**：
- 🌌 **四级宇宙探索**：从星座到卫星，层层深入每一个未知的兴趣领域。
- 💥 **引导身份塌缩**：通过预设的极端茧房身份，让用户在宇宙毁灭的震撼中，理解“认知边界”。
- 🧠 **AI 克隆体**（开发中）：一个“知音但无茧房”的数字镜像，既能提供情绪价值，又能温柔地帮你破圈。

**📅 当前进度**：
- ✅ 前端四层宇宙界面框架搭建完毕，已经完成 MVP 主链路，并已具备从星系图探索、全息预览、星系内部飞行，到进入行星观察界面的完整体验。
- 🚧 正在开发：星系内部的关联
- 🎯 下一个里程碑：MVP 上线，开放内测
